# Image_Identifiers

A comparison of three image classifier model trainers: **TensorFlow**, **Torchvision**, and **YOLO**.
Uses GPU to train models if available.

---

## Getting Started

### **Step 0: Model Preparation**
Use the provided model or train a new one using:
- _train.py

### **Step 1: Configure paths**
Select the appropriate folder paths for: 
a) the classifier model
b) test images to be identified.

### **Step 2: Run Image Classification**  
Run the model to classify images using: 
- predict_console.py
